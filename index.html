Términos de Servicio

1. Uso Aceptable

El usuario se compromete a utilizar el bot para fines lícitos. El uso de las funciones administrativas queda restringido al propietario y administradores del servidor.

2. Servicio Premium

Los pagos en Patreon son donaciones voluntarias para el mantenimiento. No son reembolsables. No se permite el spam ni acciones maliciosas bajo este estatus.

3. Responsabilidad

Caatffee no se hace responsable del mal uso por parte de los usuarios (mensajes ofensivos). Contamos con filtros preventivos, pero la conducta final es responsabilidad del usuario.

4. Gestión de Mensajes

Si un usuario decide editar un mensaje enviado a través del bot, Caatffee borrará el mensaje anterior y enviará el nuevo automáticamente por limpieza.

5. Modificaciones

Nos reservamos el derecho de actualizar estos términos para asegurar la integridad de la red Caatffee.

Política de Privacidad

1. Recopilación de Datos

Recopilamos el ID de usuario para gestionar permisos y acciones. No compartimos esta información con terceros ni extraños.

2. Almacenamiento

Al configurar acciones, los datos no se guardan permanentemente, a menos que se trate de eventos cronometrados donde la info se mantiene hasta que se cumpla la fecha.

3. Seguridad

Toda la configuración se realiza por MD para proteger la privacidad de las acciones administrativas de los servidores.

4. Derechos del Usuario

Al sacar al bot del servidor, los cambios realizados permanecen, pero el bot ya no podrá realizar nuevas acciones ni guardará ningún tipo de información residual.

5. Transparencia

Nuestras políticas están diseñadas para que el usuario siempre tenga el control sobre qué información procesa Caatffee.
